Search Page Help

To search a page on a MAC, hold down the command button and the F button at the same time. 



To search a page on a PC, hold down the  control (Cntl) button and the  F button at the same time.




Simply type the search term in the search box and hit enter/return. Any instances of the search term on the page you're on will be yellow-highlighted.
