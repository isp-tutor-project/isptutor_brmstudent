Atoms-Elements

Elements. There are many different types of atoms, called elements. Hydrogen, carbon, oxygen, sodium, and chlorine are a few different elements. An individual atom can be classified as a specific element based on the number of protons the atom has. 

The figure below is a drawing of two different elements: hydrogen and carbon. Hydrogen is the simplest element, with just one proton (p+) and one electron (e-). Unlike every other element, hydrogen doesn't have any neutrons. Oxygen is more complex (with 8 protons, 8 neutrons, and 8 electrons). Your body is made of a whole lot of atoms of these two elements! 



Atoms in their basic "elemental" forms have no "net" charge. That is, they have the same number of (negatively charged) electrons (e-) and (positively charged) protons (p+).

The elements we have discovered (so far!) are shown in Periodic Tables like this one. Elements in Periodic tables are ordered by "atomic number," which is the number of protons the element has. The atomic number/number of protons is shown in the top right corner of each box for each element. For example, as you can see in the Periodic Table, the element Sodium (Na) has 11 protons.

Ions of elements: Sometimes elements gain or lose one or more electrons. Because the atoms no longer have the same number of protons (+) and electrons (-), they have an electric charge. Atoms that have lost or gained electrons are called ions. 
  • One example of an ion is a hydrogen (H) atom that has lost an electron. It now has one proton (p+) but no electrons. So, the hydrogen ion has a positive charge. A hydrogen ion is written as H+. 
  • The "+" sign means that it has a "net" positive charge, or there are more (positively-charged) protons than (negatively-charged) electrons.
  • Another example of an ion is a Fluorine (F) atom that has gained an electron (e-). It now has one more electron than it has protons (p+), so it has a negative charge. A Fluorine ion (called Fluoride) is written as Fl-.  
  • The "-" sign means that it has a "net" negative charge, or there are more (negatively-charged) electrons than (positively-charged) protons.


Q1: An atom of the element Clorine (Cl) has a total of 18 electrons. How should this atom be represented in writing?
a) Cl

b) Cl+

c) Cl-

d) How should I know?

You are correct! Looking at the Periodic Table, you can see that Chlorine's atomic number is 17. This means that it has 17 protons. Since this Chlorine atom has 18 electrons, it has one more negatively-charged particle than positively-charged particle. So, we'd write it as Cl-.
